Project Overview

Analyzing the spatial relationship between air quality, land cover change, and population distribution across the Netherlands using satellite-derived datasets from CAMS, ESA WorldCover, and WorldPop.

Air Quality Analysis

NO2 concentration mapping using CAMS reanalysis data at 0.1 degree resolution (~11km), classified according to EU air quality standards into 5 distinct pollution levels.

Land Cover Change

Built area expansion detection between 2021 and 2023 using ESA WorldCover 10m annual land cover data.

Population Exposure

Population living in different pollution classes assessed using WorldPop counts at ~92m resolution.

Interactive WebGIS

Web-based interactive map built with OpenLayers to visualize and explore all processing results with multiple basemaps, vector layers, and click-to-inspect functionality.

Key Findings

NO2 Concentration

All 14 provinces fall within Class 1 (≤10 μg/m3) or Class 2 (10-25 μg/m3) of EU air quality standards. No provinces exceed Class 2, indicating generally acceptable air quality levels.

Population at Risk

90.2% of the Netherlands population lives in areas with Class 2 NO2 levels (10-25 μg/m3), highlighting widespread moderate pollution exposure.

Land Cover Change

Built area changes between 2021 and 2023 reveal ongoing urbanization patterns. The LCC map identifies areas of stable, gained, and lost built-up land across all provinces.

NO2 Trend (2021-2023)

Annual Mean Absolute Change (AMAC) analysis reveals decreasing NO2 levels across most regions, consistent with broader European air quality improvement trends.

Technologies Used

GDAL / QGIS

Geospatial data processing: reprojection, raster alignment, zonal statistics, and format conversions.

Python

Data analysis, raster processing with NumPy, statistical computations, and visualization with Matplotlib.

OpenLayers

Web-based interactive map with multiple basemaps, vector layers, and GIS controls.

GeoJSON / GPKG

Vector data formats for administrative boundaries and bivariate classification results.
